# Manejo de Portafolios y Asset Allocation


Este repositorio contiene notas, análisis y código desarrollados durante el seminario **Manejo de Portafolios y "Asset Allocation"**. El objetivo del curso es comprender y aplicar la teoría y práctica detrás de la construcción de portafolios de inversión eficientes bajo distintos entornos macroeconómicos.

## Descripción del Curso

A lo largo del seminario se exploran estrategias de inversión utilizando una amplia gama de instrumentos financieros. Se estudian los factores macroeconómicos que influyen en los rendimientos de los activos y se analizan métodos cuantitativos para construir portafolios balanceados y eficientes. Entre los activos considerados se encuentran:
- Bonos a tasa nominal, real y flotante
- Bonos con distintos grados de inversión
- Acciones de países desarrollados y emergentes
- Divisas
- Materias primas

Además, se implementan modelos clásicos y modernos para optimización de portafolios, evaluación de riesgo y análisis de desempeño.
